這是一款打破枯燥評量框架，結合 生成式 AI (Gemini) 與 多重沉浸式主題 的國中理化自主學習系統。

🎭 賽季主題 (Seasons)
本系統採賽季制，每一季都會更換全新的視覺與語境，讓學習保持極致的新鮮感：

⚾ 第一季：化學大聯盟 (Major League Chemistry)
情境：球員報到、春訓、季後賽。

核心：將理化知識轉化為球場戰術，AI 教練即時進行賽後分析。

🎤 第二季：黎明韓流 - 舞台選秀 (Dawn of K-Pop: The Audition)
情境：練習生評級 (A~F)、初舞台、出道選秀。

核心：每一次測驗都是一場演出，AI 導師 (Mentor) 將針對你的「舞台表現（理化知識）」給予最犀利的評語與特訓建議。

🚀 系統核心技術
BYOK (自備金鑰) 模式：

零成本營運：使用者使用個人 Gemini API Key，完全免除開發者的算力成本壓力。

全自動化題庫版控 (Auto-V-Cache)：

智慧節流：同一單元難度，全班共用 V1 題庫；挑戰第二次則自動生成 V2 考卷。API 消耗量降至最低。

防幻覺知識封印 (Science Guard)：

嚴謹科學：無論主題如何變換（棒球或韓流），AI 在解釋化學公式與定義時，絕對保持 100% 的科學正確性，禁止牽強比喻。

跨裝置身分綁定：

一鍵報到：支援年級、班級、座號、藝名（暱稱）設定，下載戰報即刻與老師對接成績。
